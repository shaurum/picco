# Правила сборки контроллера

При сборке контроллера необходимо придерживаться следующих правил:

- для модульной группы, включающей в себя Модуль основной GMB, сборку необходимо начать с установки Модуля ввода питания SPPM, затем монтировать Модуль основной GMB, после которого добавлять модули расширения.
- для модульных групп, не содержащих Модуль основной GMB, сборку необходимо начать с установки Модуля оконечного SPTM, затем монтировать Модуль ввода питания SPPM, после которого добавляются модули расширения.  
- при расширении контроллера последующими группами или подключения по схеме [«Кольцо»](assembly.md#_3), в конце группы ставится модуль оконечный.
- модули расширения в группе ставятся в количестве, рассчитываемом в соответствии с разделом [Организация питания](power_organization.md#организация питания).
- на первый и последний модули в группе в обязательном порядке ставится заглушка.

!!! note "Примечание"
    На этапе сборки рекомендуется предусмотреть резервирование контроллера, что обеспечит непрерывную работу системы в случае отказа основного контроллера.

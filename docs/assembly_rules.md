# Правила сборки контроллера

При сборке контроллера необходимо придерживаться следующих правил:

- для модульной группы, включающей в себя основной модуль, сборку необходимо начать с установки [Модуля ввода питания SPPM](SPPM.md), затем монтировать основной модуль, после которого добавлять модули расширения.
- для модульных групп, не содержащих основной модуль, сборку необходимо начать с установки оконечного модуля, затем монтировать [Модуль ввода питания SPPM](SPPM.md), после которого добавляются модули расширения.  

    !!! note "Примечание"
        Выбор оконечного модуля осуществляется следующим образом:

        - Если модульные группы связываются при помощи Ethernet-кабеля с патч-кордами **RJ-45**, устанавливается [Модуль оконечный SPTM](SPTM.md)
        - Если модульные группы связываются при помощи **оптоволоконного кабеля**, устанавливается [Модуль оконечный SFP](SPSFP.md)
    
- при расширении контроллера последующими группами или подключения по схеме [«Кольцо»](assembly.md#_2), в конце группы ставится модуль оконечный.
- [Модули ввода питания SPPM](SPPM.md) ставятся в количестве, рассчитываемом в соответствии с разделом [Организация питания](power_organization.md#организация питания).
- на первый и последний модули в группе в обязательном порядке ставится заглушка.

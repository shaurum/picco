# Правила сборки контроллера

При сборке контроллера необходимо придерживаться следующих правил:

- при наличии в собираемой группе модуля основного GMB, группа собирается в следующем порядке: модуль ввода питания SPPM, модуль основной GMB, модули расширения.
- при отсутствии в собираемой группе модуля основного GMB, группа собирается в следующем порядке: модуль оконечный SPTM, модуль ввода питания SPPM, модули расширения.
- при расширении контроллера последующими группами или подключения по схеме «Кольцо», в конце группы ставится модуль оконечный.
- модули расширения в группе ставятся в количестве, рассчитываемом в соответствии с п. 5.2.
- на первый и последний модули в группе в обязательном порядке ставится заглушка.

!!! note "Примечание"
    На этапе сборки рекомендуется предусмотреть резервирование контроллера, что обеспечит непрерывную работу системы в случае отказа основного контроллера.

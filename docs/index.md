## [Руководство пользователя](basic_nformation.md)

Данное руководство содержит сведения о конструкции, принципе действия, технических характеристиках изделия, а также указания, необходимые для правильной и безопасной эксплуатации программируемого логического контроллера.

Руководство предназначено для эксплуатационного персонала и инженеров-проектировщиков АСУ ТП. **К работе с контроллером допускаются только лица, изучившие настоящее руководство.**
